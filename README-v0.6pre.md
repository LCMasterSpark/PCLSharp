# Plain Craft Launcher Sharp v0.6pre

Plain Craft Launcher Sharp，简称 **PCL Sharp / PCL#**，是一个用 C# WPF 重新实现 Plain Craft Launcher 体验的实验性重构项目。

## 重要说明

- 这是一个 **未完成的实验预览版**，仅用于交流、测试和展示当前重构进度。
- 该版本不是原 Plain Craft Launcher 的官方发布版，也不建议作为日常主力启动器使用。
- 当前功能仍在迁移中，部分页面、下载、登录、启动补全、崩溃分析、资源管理等能力可能缺失、不完整或行为不稳定。
- 运行前请自行备份 Minecraft 目录、实例、整合包和重要配置。

## 当前版本

- 版本号：v0.6pre
- 程序名：Plain Craft Launcher Sharp
- 简称：PCL Sharp / PCL#
- 技术栈：.NET 10、C#、WPF、MVVM
- 发布类型：Windows x64 自包含运行时包

## 当前重构范围

本预览版已经具备以下基础能力：

- VS2022 风味主题。
- 启动、下载、实例、设置、更多等主要页面框架。
- 本地 Minecraft 文件夹与版本扫描。
- 实例选择、实例管理、Mod 列表展示等基础交互。
- 启动链路的基础参数生成与本地检查框架。
- JSON 设置系统、日志、路径服务、加载器模型和测试体系。
- 部分旧版 PCL 图片资源已内置到新项目中，用于图标和界面展示。

仍未完成或仍需继续打磨的内容包括：

- 完整的一比一启动体验。
- 完整微软登录、第三方验证服务器登录与缓存迁移。
- 联网下载补全与资源校验。
- 下载页完整生态功能。
- 崩溃分析、游戏进程监控和异常提示细节。
- 更多页面交互、动画、细节 UI 和兼容性处理。

## 使用建议

1. 解压后运行 `PlainCraftLauncherSharp.exe`。
2. 首次启动后可在实例或启动页面添加/切换 Minecraft 文件夹。
3. 如果遇到界面错位、功能缺失、启动失败或扫描异常，请把截图和日志一起反馈。
4. 日志目录通常位于 `%LOCALAPPDATA%\Plain Craft Launcher Sharp\Logs`。

## 重构目标

本项目的目标不是简单逐行翻译旧代码，而是在功能一比一还原的前提下，将旧版中较强的全局状态和 UI 耦合逐步拆成可测试、可维护的 C# MVVM 服务结构。

当前版本只是阶段性快照，后续仍会继续推进功能补齐、交互打磨、代码拆分和稳定性测试。
